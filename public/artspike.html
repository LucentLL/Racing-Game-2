How faithful is this? The mock uses the game's real primitives — 18 px ground tiles, stroked asphalt with dashed lane lines, flat-fill top-down cars, procedural shingle roofs — so proportions and layering match what the engine already draws. The cel treatment is per-shape (outlines + 2-tone bands + hard cast shadows), which is exactly how it would land in-engine, one renderer at a time. In the real game the dither would be baked into each tile once at boot (not a per-frame pass), so it costs nothing to draw. No architectural changes — it's an art pass over ~12–15 renderer files plus a shared palette module.
