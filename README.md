# Driver City — GBC Racer

A Game Boy Color-aesthetic top-down racing game set in 1999 Charlotte, NC.

## Current Build
`driver_city_charlotte_v8_99_126_89.html` — open in a browser to play.

## Structure
Currently delivered as a single HTML file with inline JS/CSS. Sprite assets
(PNGs, WAVs) live alongside it and are loaded at runtime via raw GitHub URLs.

Migration to a folder structure is in progress.
